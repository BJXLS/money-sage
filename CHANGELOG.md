## [0.1.0] - 2025-01-XX

### 🎉 首次发布

这是 MoneySage 的首个公开版本！

### ✨ 新功能

#### 核心功能

- **AI 快速记账** - 使用自然语言快速记录收支
  - 自动识别金额、分类、日期
  - 支持批量输入多条记录
  - 智能解析中文描述
- **仪表盘** - 财务状况一目了然
  - 月度收支汇总
  - 收支趋势图（3/6/12 个月）
  - 支出分布饼图
  - 最近交易列表
  - 预算执行监控
- **交易记录管理** - 完整的记账功能
  - 添加、编辑、删除交易
  - 多条件筛选和搜索
  - 分页浏览历史记录
  - 按多种字段排序
- **分类管理** - 灵活的分类系统
  - 预设 12 个常用分类
  - 支持自定义分类
  - 分类图标和颜色配置
- **预算管理** - 帮助控制支出
  - 按分类设置预算
  - 支持周/月/年度预算
  - 实时预算进度监控
  - 超支提醒
- **统计分析** - 深入了解财务状况
  - 月度趋势分析
  - 分类支出统计
  - 多维度图表展示
- **数据导入导出** - 方便的数据管理
  - CSV 格式导入
  - CSV 格式导出
  - 数据备份和迁移

#### 技术特性

- **本地数据库** - SQLite 存储，隐私安全
- **现代化界面** - 深色主题设计
- **响应式布局** - 适应不同屏幕尺寸
- **高性能** - Rust + Vue 技术栈
- **跨平台框架** - 基于 Tauri 2.0

### 🎨 用户界面

- 深色主题设计，减少眼睛疲劳
- 流畅的动画过渡效果
- 直观的导航和操作
- 美观的图表和数据可视化

### 📦 安装包

- Windows MSI 安装包
- Windows NSIS 安装包
- 支持 Windows 10/11 x64

### 🔧 技术栈

- **前端**: Vue 3 + TypeScript + Vite + Element Plus
- **后端**: Rust + Tauri 2.0
- **数据库**: SQLite + sqlx
- **图表**: ECharts
- **AI**: 支持多种 LLM API

## 版本说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

版本格式：`主版本号.次版本号.修订号`

- **主版本号**: 做了不兼容的 API 修改
- **次版本号**: 做了向下兼容的功能性新增
- **修订号**: 做了向下兼容的问题修正
