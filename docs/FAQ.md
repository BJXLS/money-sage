## 安装和启动

### Q: 支持哪些操作系统？

**A**: 目前仅支持 Windows 10 和 Windows 11（64 位）。未来计划支持 macOS 和 Linux。

### Q: 安装时提示需要 WebView2，这是什么？

**A**: WebView2 是 Microsoft Edge 的 Web 渲染引擎。

- **Windows 11**: 系统自带，无需额外安装
- **Windows 10**: 安装程序会自动下载安装
- 手动下载: https://developer.microsoft.com/microsoft-edge/webview2/

### Q: MSI 和 NSIS 安装包有什么区别？

**A**:

- **MSI**: Windows 原生格式，适合企业环境和批量部署
- **NSIS**: 安装向导更友好，文件更小，推荐个人用户使用

两者功能完全相同，选择任意一个即可。

### Q: 安装后无法启动怎么办？

**A**: 请尝试：

1. 确认系统是 Windows 10/11 64 位
2. 检查是否有杀毒软件拦截
3. 右键以管理员身份运行
4. 检查 WebView2 是否正确安装
5. 查看日志文件（位于用户文档目录）

### Q: Windows Defender 提示不安全？

**A**: 这是因为应用没有代码签名证书（需要购买）。应用是安全的，你可以：

1. 点击"更多信息"
2. 选择"仍要运行"
3. 或者从源码自行构建

## 使用问题

### Q: 如何开始记账？

**A**:

1. 点击右上角"快速记账"按钮
2. 输入自然语言描述，如"今天午饭花了 35 元"
3. AI 自动识别后确认保存
4. 或者在"记账"页面手动添加

### Q: 支持的自然语言格式是什么？

**A**: 支持中文日常表达，例如：

- "今天中午吃饭花了 38 元"
- "昨天坐地铁 12 元"
- "收到工资 5000 元"
- "买书花了 89"（默认今天、自动识别分类）

### Q: AI 识别错误怎么办？

**A**:

- 在确认界面可以直接修改识别结果
- 修改金额、分类、日期、描述等任何字段
- 确认无误后再保存

### Q: 如何修改已记录的交易？

**A**:

1. 进入"记账"页面
2. 找到要修改的记录
3. 点击"编辑"按钮
4. 修改后保存

### Q: 如何设置预算？

**A**:

1. 进入"预算设置"页面
2. 点击"添加预算"
3. 选择分类、设置金额和周期
4. 保存后在仪表盘可以看到预算执行情况

### Q: 预算超支了会怎样？

**A**:

- 在仪表盘会显示超支提醒（红色）
- 预算进度条会超过 100%
- 不会阻止继续记账，只是提醒

### Q: 可以添加自定义分类吗？

**A**:

- 可以！进入"分类管理"页面
- 点击"添加分类"
- 设置名称、图标、颜色、类型
- 系统预设分类不能删除，但可以删除自定义分类

## 数据管理

### Q: 数据存储在哪里？

**A**:

- 存储在本地 SQLite 数据库
- Windows 位置: `C:\Users\[用户名]\AppData\Roaming\com.money-note.app\money_note.db`
- 数据完全本地化，不会上传云端

### Q: 如何备份数据？

**A**: 两种方式：

1. **导出 CSV**（推荐）
   - 进入"导入导出"页面
   - 点击"导出数据"
   - 保存 CSV 文件
2. **复制数据库文件**
   - 找到 `money_note.db` 文件
   - 复制到安全位置

### Q: 如何恢复数据？

**A**:

1. 使用 CSV 导入功能恢复交易记录
2. 或直接替换数据库文件（需要关闭应用）

### Q: 更换电脑如何迁移数据？

**A**:

1. 在旧电脑导出 CSV
2. 在新电脑安装 MoneySage
3. 导入 CSV 文件
4. 预算和分类需要重新设置（或复制数据库文件）

### Q: 误删数据可以恢复吗？

**A**:

- 应用内没有回收站功能
- 如果有备份的 CSV 或数据库文件可以恢复
- 建议定期备份重要数据

### Q: 支持多设备同步吗？

**A**:

- 当前版本不支持
- 计划在未来版本添加云同步功能
- 目前可以通过手动导入导出实现

## AI 功能

### Q: AI 功能必须使用吗？

**A**:

- 不是必须的
- 可以选择手动添加交易记录
- AI 功能主要是为了提高效率

### Q: 需要自己的 API 密钥吗？

**A**:

- 是的，需要配置 LLM API 密钥
- 点击右上角设置按钮配置
- 支持 OpenAI、Claude 等兼容 API

### Q: 支持哪些 AI 模型？

**A**:

- OpenAI (GPT-3.5, GPT-4)
- Anthropic Claude
- 其他兼容 OpenAI API 格式的模型
- 本地模型（如果有 API 接口）

### Q: AI 会上传我的财务数据吗？

**A**:

- AI 功能会将你输入的**描述文本**发送到 LLM 服务
- 不会上传你的历史数据、账户余额等
- 建议不要在描述中包含敏感信息（如银行账号、密码等）

### Q: API 调用会产生费用吗？

**A**:

- 使用商业 API（如 OpenAI）会产生费用
- 费用取决于使用频率和模型
- 每次快速记账大约消耗少量 tokens（几分钱）
- 可以查看 API 提供商的定价

### Q: AI 识别准确率如何？

**A**:

- 对于标准表述准确率很高（90%+）
- 建议在确认界面检查结果
- 可以随时手动修改

### Q: 没有 API 密钥可以使用吗？

**A**:

- 可以，但无法使用 AI 快速记账功能
- 其他功能（仪表盘、统计、预算等）完全正常使用
- 可以手动添加交易记录

## 技术问题

### Q: 应用占用多少空间？

**A**:

- 安装包: ~50-80MB
- 安装后: ~100MB
- 数据库: 根据记录数量，通常 < 10MB

### Q: 需要联网吗？

**A**:

- 基本功能（记账、统计等）不需要联网
- AI 快速记账功能需要联网调用 API
- 无网络时可以手动添加记录

### Q: 支持多用户吗？

**A**:

- 当前版本不支持
- 每个 Windows 用户账户有独立的数据
- 未来计划添加多账本功能

### Q: 数据库加密吗？

**A**:

- 当前版本数据库未加密
- 依赖操作系统的用户权限保护
- 建议使用 Windows BitLocker 加密磁盘
- 未来计划添加数据库加密功能

### Q: 如何查看日志？

**A**:
日志位置：

- Windows: `C:\Users\[用户名]\AppData\Roaming\com.money-note.app\logs\`

### Q: 应用更新会丢失数据吗？

**A**:

- 不会，数据和应用是分离的
- 更新前建议备份（导出 CSV）
- 数据库位置不会改变

### Q: 可以在虚拟机中运行吗？

**A**:

- 可以
- 需要确保虚拟机支持 WebView2
- 性能可能略有影响

### Q: 支持便携版吗？

**A**:

- 当前版本不支持
- 可以在 GitHub Issues 中请求此功能

## 性能和限制

### Q: 最多支持多少条交易记录？

**A**:

- SQLite 理论上无限制
- 测试过 10 万+ 记录，性能良好
- 大数据量时搜索和筛选可能略慢

### Q: 可以导入大量历史数据吗？

**A**:

- 可以通过 CSV 批量导入
- 建议分批导入（每次 < 1000 条）
- 导入前确保 CSV 格式正确
