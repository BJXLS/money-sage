# 记账本 - MoneyNote 使用说明

## 功能概览

这是一个现代化的个人记账软件，支持Windows系统，提供简约时尚的用户界面和完整的记账功能。

## 主要功能

### 1. 仪表盘 📊
- **收支概览**: 显示本月收入、支出、结余和交易笔数
- **收支趋势图**: 可选择查看近3/6/12个月的收支趋势
- **支出分布饼图**: 按分类显示当月支出占比
- **最近交易**: 显示最新的8条交易记录
- **预算执行**: 显示各分类的预算使用情况

### 2. 交易记录管理 💰
- **添加记录**: 点击"记一笔"快速添加收入或支出
- **筛选搜索**: 按类型、分类、日期范围筛选记录
- **编辑删除**: 支持修改和删除已有记录
- **分页显示**: 支持大量数据的分页浏览
- **排序功能**: 按日期、金额等字段排序

### 3. 分类管理 🏷️
系统预设了以下分类：

**支出分类**:
- 🍽️ 餐饮 (红色)
- 🚗 交通 (青色) 
- 🛍️ 购物 (蓝色)
- 🎬 娱乐 (绿色)
- 🏠 住房 (黄色)
- ⚕️ 医疗 (紫色)
- 📚 教育 (橙色)
- 💰 其他支出 (灰色)

**收入分类**:
- 💼 工资 (绿色)
- 💪 兼职 (蓝色)
- 📈 投资 (紫色)
- 💰 其他收入 (绿色)

用户可以自定义添加新分类或删除非系统分类。

### 4. 预算管理 📈
- **设置预算**: 为各支出分类设置月度/年度预算
- **进度监控**: 实时显示预算使用情况
- **超支提醒**: 当支出接近或超过预算时进行提醒
- **预算分析**: 在仪表盘显示预算执行详情

### 5. 统计分析 📊
- **月度趋势**: 查看历史收支变化趋势
- **分类统计**: 按分类分析支出分布
- **时间筛选**: 选择不同时间段进行分析
- **图表展示**: 直观的图表呈现数据

### 6. 数据导入导出 📁
- **CSV导入**: 批量导入交易记录
- **CSV导出**: 导出数据进行备份或分析
- **格式支持**: 支持标准CSV格式

## 使用步骤

### 首次使用
1. 启动应用，系统会自动创建数据库
2. 默认分类已预设，可直接开始记账
3. 点击"记一笔"添加第一条交易记录

### 日常记账
1. **快速记账**: 点击右上角"记一笔"按钮
2. **选择类型**: 收入或支出
3. **填写信息**: 金额、分类、日期、描述等
4. **保存记录**: 点击确定完成记账

### 查看数据
1. **仪表盘**: 查看整体财务状况
2. **交易记录**: 查看详细的交易历史
3. **统计分析**: 分析消费模式和趋势

### 预算管理
1. 进入"预算管理"页面
2. 为各支出分类设置预算金额
3. 选择预算周期（周/月/年）
4. 系统会自动跟踪预算执行情况

## 数据安全

- **本地存储**: 所有数据存储在本地SQLite数据库
- **自动备份**: 建议定期使用导出功能备份数据
- **隐私保护**: 数据不会上传到云端，保护个人隐私

## 系统要求

- 操作系统: Windows 10 或更高版本
- 内存: 最少2GB RAM
- 磁盘空间: 约100MB

## 技术特性

- **现代化界面**: 使用Element Plus组件库
- **响应式设计**: 支持不同屏幕尺寸
- **高性能**: Rust后端 + Vue前端
- **跨平台**: 基于Tauri框架

## 快捷操作

- **添加记录**: Ctrl + N
- **搜索**: Ctrl + F
- **刷新数据**: F5

## 注意事项

1. 删除分类前请确保该分类下无交易记录
2. 系统分类无法删除，仅可删除自定义分类
3. 建议定期导出数据进行备份
4. 金额支持到小数点后两位

## 常见问题

**Q: 如何备份数据？**
A: 使用"导入导出"功能导出CSV文件进行备份。

**Q: 可以修改已有的交易记录吗？**
A: 可以，在交易记录页面点击编辑按钮即可修改。

**Q: 预算超支了怎么办？**
A: 系统会在仪表盘显示超支提醒，您可以调整预算或控制支出。

**Q: 支持多币种吗？**
A: 当前版本仅支持人民币（￥）。
