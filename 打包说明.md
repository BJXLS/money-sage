# MoneyNote Windows 安装包制作说明

## 快速开始

### 方法一：使用自动化脚本（推荐）

1. **双击运行批处理文件**：
   ```
   build-installer.bat
   ```

2. **或运行 PowerShell 脚本**：
   ```powershell
   .\build-installer.ps1
   ```

脚本会自动检查环境、安装依赖并构建安装包。

### 方法二：手动命令行

```bash
# 1. 安装依赖
npm install

# 2. 构建安装包
npm run build:installer
```

## 环境要求

### 必需工具
- **Node.js 18+** - JavaScript 运行环境
- **Rust** - Tauri 后端语言

### 安装包构建工具（至少一个）
- **WiX Toolset 3.11** - 构建 MSI 安装包
- **NSIS** - 构建 EXE 安装包

## 安装步骤

### 1. 安装 Node.js
- 访问 https://nodejs.org/
- 下载并安装 LTS 版本

### 2. 安装 Rust
- 访问 https://rustup.rs/
- 按照指示安装

### 3. 安装 WiX Toolset（MSI 包）
- 访问 https://github.com/wixtoolset/wix3/releases
- 下载 `wix311.exe` 并安装
- 确保添加到 PATH 环境变量

### 4. 安装 NSIS（EXE 包）
- 访问 https://nsis.sourceforge.io/Download
- 下载并安装最新版本
- 确保添加到 PATH 环境变量

## 输出文件

构建成功后，安装包位于：
```
src-tauri/target/release/bundle/
├── msi/
│   └── money-sage_0.1.0_x64_zh-CN.msi
└── nsis/
    └── money-sage_0.1.0_x64-setup.exe
```

## 两种安装包的区别

| 类型 | 文件格式 | 特点 | 适用场景 |
|------|----------|------|----------|
| MSI | .msi | Windows 原生格式，支持企业部署 | 企业环境、批量部署 |
| NSIS | .exe | 安装向导更灵活，文件更小 | 个人用户、一般分发 |

## 常见问题

### Q1: 构建时提示找不到 WiX
**解决方案**：
1. 确认已安装 WiX Toolset 3.11（不是 4.x）
2. 检查 PATH 环境变量是否包含 WiX 安装目录
3. 重启命令行工具

### Q2: 构建时提示找不到 NSIS
**解决方案**：
1. 确认已安装 NSIS
2. 检查 PATH 环境变量是否包含 NSIS 安装目录
3. 重启命令行工具

### Q3: 安装包无法在其他电脑上运行
**解决方案**：
1. 确保目标电脑安装了 Microsoft Edge WebView2
2. 如果仍有问题，可能需要代码签名证书

### Q4: 想要修改应用图标
**解决方案**：
1. 替换 `src-tauri/icons/` 目录下的图标文件
2. 重新构建安装包

## 版本发布流程

1. **更新版本号**：
   - 修改 `package.json` 中的 `version`
   - 修改 `src-tauri/tauri.conf.json` 中的 `version`
   - 修改 `src-tauri/Cargo.toml` 中的 `version`

2. **构建安装包**：
   ```bash
   npm run build:installer
   ```

3. **测试安装包**：
   - 在干净的 Windows 环境中测试安装
   - 确认应用功能正常

4. **分发**：
   - 上传到发布平台
   - 提供详细的更新说明

## 高级配置

### 代码签名
在 `src-tauri/tauri.conf.json` 中配置：
```json
{
  "bundle": {
    "windows": {
      "certificateThumbprint": "YOUR_CERTIFICATE_THUMBPRINT",
      "timestampUrl": "http://timestamp.sectigo.com"
    }
  }
}
```

### 自定义安装界面
可以为 NSIS 添加自定义图片：
```json
{
  "bundle": {
    "windows": {
      "nsis": {
        "headerImage": "path/to/header.bmp",
        "sidebarImage": "path/to/sidebar.bmp"
      }
    }
  }
}
```
