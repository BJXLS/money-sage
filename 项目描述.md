# MoneySage 项目描述

## 📋 项目概览

**MoneySage** 是一款现代化的智能记账桌面应用，采用 AI 辅助快速记账技术。该项目基于 Tauri 2.0 框架开发，结合了 Vue 3 前端和 Rust 后端，为用户提供高性能、隐私安全的本地记账体验。

## 🎯 主要功能

### 1. 🤖 AI 智能记账
- **自然语言处理**：支持用自然语言描述收支情况，AI 自动识别金额、分类、日期和类型
- **批量记账**：支持一次输入多条记录，自动批量解析
- **智能识别**：例如输入"今天中午在餐厅花了38元吃午饭"，AI 自动识别为餐饮支出38元

### 2. 📊 数据可视化仪表盘
- **收支汇总**：实时显示本月收入、支出、结余和交易笔数
- **趋势分析**：可视化展示近3/6/12个月的收支变化趋势
- **支出分布**：饼图显示各分类支出占比
- **预算监控**：实时显示预算使用情况和进度

### 3. 💰 交易记录管理
- **快速添加**：一键"记一笔"功能
- **智能筛选**：按类型、分类、日期范围筛选
- **灵活编辑**：支持修改和删除记录
- **分页浏览**：高效管理大量历史数据

### 4. 🏷️ 分类管理
- **预设分类体系**：
  - 支出：餐饮、交通、购物、娱乐、住房、医疗、教育、其他
  - 收入：工资、兼职、投资、其他
- **自定义分类**：支持添加自定义分类、图标和颜色

### 5. 📈 预算管理
- **灵活设置**：为各支出分类设置周/月/年度预算
- **实时监控**：动态显示预算使用百分比和剩余额度
- **超支提醒**：接近或超过预算时智能提醒

### 6. 📊 统计分析
- **趋势图表**：查看历史收支变化趋势
- **分类统计**：深入分析各分类支出分布
- **数据洞察**：帮助发现消费模式，优化财务规划

### 7. 📁 数据导入导出
- **CSV 导入/导出**：支持批量导入交易记录和数据备份
- **标准格式**：兼容常见表格软件

## 🛠️ 技术栈

### 前端技术
- **核心框架**: Vue 3.5.13 (Composition API)
- **构建工具**: Vite 6.0.3
- **UI 框架**: Element Plus 2.8.5
- **状态管理**: Pinia 2.2.6
- **图表库**: ECharts 5.5.0 + vue-echarts 7.0.3
- **工具库**:
  - VueUse 11.2.0 (Vue 组合式工具集)
  - Day.js 1.11.13 (日期处理)
  - PapaParse 5.4.1 (CSV 处理)
- **开发语言**: TypeScript 5.6.2

### 后端技术
- **核心框架**: Tauri 2.0 (桌面应用框架)
- **编程语言**: Rust (Edition 2021)
- **数据库**: SQLite + sqlx 0.8
- **异步运行时**: Tokio 1.0
- **HTTP 客户端**: reqwest 0.11 (AI API 调用)
- **序列化**: serde 1.0 + serde_json 1.0
- **错误处理**: anyhow 1.0 + thiserror 1.0
- **其他工具**:
  - chrono 0.4 (日期时间处理)
  - uuid 1.0 (唯一标识符)
  - csv 1.1 (CSV 处理)

### 开发工具
- **包管理**: npm (前端) + Cargo (后端)
- **代码格式化**: Prettier + rustfmt
- **自动导入**: unplugin-auto-import + unplugin-vue-components

## 💾 数据存储

### 数据库设计
- **数据库类型**: SQLite (本地文件数据库)
- **数据库文件**: `money_note.db`
- **主要数据表**:
  - `transactions` - 交易记录
  - `categories` - 分类信息
  - `budgets` - 预算设置

### 数据特点
- **本地存储**: 所有数据存储在本地，保护用户隐私
- **轻量级**: SQLite 数据库文件小，运行效率高
- **无需网络**: 完全离线运行，不依赖云端服务

## 🔧 开发构建

### 开发环境
- **Node.js**: 18+
- **Rust**: 通过 rustup 安装
- **平台支持**: 主要支持 Windows (可扩展到 macOS/Linux)

npm run build
